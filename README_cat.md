# Mapa lliscant de cartografia de fanerògames marines de Mataró

Aquest repositori allotja un **mapa interactiu amb lliscador** dissenyat per comparar visualment dues cartografies de fanerògames marines a l’àrea costanera de **Mataró**:

- **2010**: INTECSA-INARSA, *Cartografia de les comunitats de fanerògames marines situades davant del terme municipal de Mataró (Maresme)*. Diputació de Barcelona, Àrea de Medi Ambient.
- **2025**: [Mapa dels hàbitats marins a Catalunya](https://mediambient.gencat.cat/ca/05_ambits_dactuacio/patrimoni_natural/sistemes_dinformacio/habitats/habitats-marins/projecte-mapa-dels-habitats-marins-de-catalunya/). Generalitat de Catalunya, Departament de Medi Ambient i Sostenibilitat.

A més, inclou una sèrie de dades de referència per avaluar l’exactitud d’ambdues cartografies, recopilades de manera col·laborativa per:

- Grup d’Ecologia d’Angiospermes Marines (GEAM), [Institut Espanyol d’Oceanografia (IEO-CSIC)](https://www.ieo.es).
- [Centre d’Estudis Avançats de Blanes (CEAB-CSIC)](https://www.ceab.csic.es).
- [Universitat de Barcelona](https://www.ub.edu/portal/web/dp-beeca-es/ecologia1).
- [Escola de Mar de Badalona](https://www.badalona.cat/ca/serveis-ajuntament/mediambient-i-sostenibilitat/edudacio-ambiental/escola-del-mar).

## Objectiu
La interfície amb lliscador permet als usuaris explorar les diferències espacials entre els mapes d’hàbitats desplaçant una barra vertical que revela una capa sobre l’altra, facilitant l’avaluació visual dels canvis en els criteris de cartografia i en l’extensió de les praderies de fanerògames marines. Capes addicionals de dades de referència es poden activar o desactivar mitjançant el menú situat a la cantonada superior esquerra.

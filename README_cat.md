# Visualització per comparar cartografies de fanerògames marines a Mataró

Aquest repositori allotja un **mapa interactiu** dissenyat per comparar visualment dues cartografies de fanerògames marines a l’àrea costanera de **Mataró**:

- **2010**: INTECSA-INARSA, Cartografia de les comunitats de fanerògames marines situades davant del terme municipal de Mataró (Maresme). Diputació de Barcelona, Àrea de Medi Ambient.
- **2025**: [Mapa dels hàbitats marins de Catalunya](https://mediambient.gencat.cat/ca/05_ambits_dactuacio/patrimoni_natural/sistemes_dinformacio/habitats/habitats-marins/projecte-mapa-dels-habitats-marins-de-catalunya/). Generalitat de Catalunya, Departament de Medi Ambient i Sostenibilitat.

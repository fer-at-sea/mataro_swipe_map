# Mataró Seagrass Cartography Swipe Map

This repository hosts an interactive **swipe map** designed to visually compare two seagrass cartographies for the coastal area of **Mataró (Catalunya)**:

- **[Atlas de las praderas marinas de España 2015](https://www.ieo.es/es/atlas-praderas-marinas)** (Carto Herbassars Generalitat de Catalunya & Instituto Español de Oceanografía)
- **[Mapa dels hàbitats marins a Catalunya](https://mediambient.gencat.cat/ca/05_ambits_dactuacio/patrimoni_natural/sistemes_dinformacio/habitats/habitats-marins/projecte-mapa-dels-habitats-marins-de-catalunya/)** (Generalitat de Catalunya, Medi Ambient i Sostenibilitat) 

The swipe interface allows users to explore spatial differences between both cartographies by sliding a vertical bar that reveals one layer over the other.

## Objectives
- Provide an intuitive comparison of habitat boundaries and classifications.
- Facilitate a visual assessment of changes in mapping criteria, precision, and spatial extent.

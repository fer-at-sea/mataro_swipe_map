# Mataró Seagrass Cartography Swipe Map

This repository hosts an interactive **swipe map** designed to visually compare two seagrass cartographies for the coastal area of **Mataró (Catalunya)**:

- **2010 Cartografia de les comunitats de les fanerògames marines situades davant del terme municipal de Mataró as presented in [Herbassars](https://agricultura.gencat.cat/ca/serveis/cartografia-sig/bases-cartografiques/pesca-maritima-proteccio-litoral/herbassars-praderies-fanerogames-marines/)** (Generalitat de Catalunya, Departament d'Agricultura, Ramaderia, Pesca i Alimentació) in 2010.
- **2025 [Mapa dels hàbitats marins a Catalunya](https://mediambient.gencat.cat/ca/05_ambits_dactuacio/patrimoni_natural/sistemes_dinformacio/habitats/habitats-marins/projecte-mapa-dels-habitats-marins-de-catalunya/)** (Generalitat de Catalunya, Departament de Medi Ambient i Sostenibilitat) 

The swipe interface allows users to explore spatial differences between both cartographies by sliding a vertical bar that reveals one layer over the other.

## Objectives
- Provide an intuitive comparison of habitat boundaries and classifications.
- Facilitate a visual assessment of changes in mapping criteria and seagrass meadow extent.
